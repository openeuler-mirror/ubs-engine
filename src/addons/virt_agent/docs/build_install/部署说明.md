# 部署说明

## 1 概述

virt agent的发布件包含：运行包

- 运行包：主要用于在生产环境中与周边ubfm等模块交互，提供ub池化资源管理能力

virt agent的发布件，由以下两个 RPM 包组成：

| RPM包                                                   | 说明                                   |
| ------------------------------------------------------- | -------------------------------------- |
| ubs-engine-virtagent-<version>-<release>.aarch64.rpm              | 主程序包       |
| python3-ubs-engine-<version>.rpm  | python外层接口包 |

## 2 安装开发包

### 2.1 环境要求

业务开发环境中部署即可，无特殊要求，Virt Agent作为UBS插件，部署依赖完整安装的 ubs-engine。

### 2.2 系统要求

* <b>操作系统：</b> openEuler 24.03 LTS 或更高版本
* <b>CPU架构：</b> aarch64
* <b>内存：</b> ≥ 64GB
* <b>磁盘：</b> SSD，IOPS 500MB/s
* <b>芯片互联：</b> UB
* <b>网卡：</b>可选依赖（可选使用TCP辅助UB建链，默认采用UB自举建链）
* <b>用户权限：</b> 安装与管理需 <code>root</code> 权限

### 2.3 节点规划

* 所有集群节点均需安装 <code>ubs-engine</code> 主包。

* 建议提前规划节点 IP 地址列表（用于 TCP 模式）

### 2.4 依赖运行时库

运行依赖以下系统库，通常由包管理器自动安装：

* <code>glibc</code>, <code>libstdc++</code>, <code>libgcc</code>

* <code>libxml2</code>

* <code>numactl-libs</code>

* <code>libboundscheck</code>

* <code>ubs-hcom</code>

* <code>systemd</code>

### 2.5 RPM 包安装步骤

```bash
# 安装主程序包
sudo dnf install -y ubs-engine-virtagent-1.0.0-1.aarch64.rpm

# 安装python外层接口包
sudo dnf install -y python3-ubs-engine-1.0.0-1.noarch.rpm
```

> <p style="font-family: SimSun; font-size: 16px;">若使用 <code>rpm</code> 命令，请注意依赖顺序：</p>
> ```bash
> sudo rpm -ivh ubs-engine-virtagent-1.0.0-1.aarch64.rpm
> sudo rpm -ivh python3-ubs-engine-1.0.0-1.noarch.rpm
> ```

### 3.6 安装结果

- Virt-Agent 主程序安装结果：

| 路径               | 用途             |
|------------------|----------------|
| /etc/ubse/       | 配置目录           |
| /usr/lib/python3.11/site-packages/ubse/   | python sdk配置目录 |
| /usr/lib64/    | 主程序库目录         |


> 完整权限说明见《安全设计》文档

- virt-agent 客户端运行时库安装结果：

| 文件                                      | 其它说明                                          |
|-----------------------------------------| ------------------------------------------------- |
| `/usr/lib64/libstrategy.so`             | 二进制动态库实体          |
| `/usr/lib64/libubs-virt-agent.so`       | 软链接，指向 `/usr/lib64/libubs-virt-agent.so.1`          |
| `/usr/lib64/libvirtagent.so`            | 二进制动态库实体          |
| `/usr/lib64/libubs-virt-agent.so.1.0.0` | 二进制动态库实体                                  |
| `/usr/lib64/libubs-virt-agent.so.1`     | 软链接，指向 `/usr/lib64/libubs-virt-agent.so.1.0.0` |

> 完整权限说明见《安全设计》文档

## 3 服务启动与停止
### 3.1 启动virt agent服务
作为 ubs engine 插件，依赖 ubs进程

```bash
sudo systemctl start ubse
sudo systemctl enable ubse
```

### 3.2 停止服务

```bash
sudo systemctl stop ubse
```

### 3.3 查看状态

```bash
sudo systemctl status ubse
```

### 3.4 开机自启

```bash
sudo systemctl enable ubse
```

## 4 验证部署

### 4.1 检查服务状态

```bash
systemctl is-active ubse  # 应输出 "active"
```

### 4.2 查看日志

```bash
#方式一：
journalctl -u ubse -f

#方式二：参看日志文件
/var/log/ubse/vm_plugin.log
```

## 5 升级与卸载

### 5.1 升级

```bash
sudo dnf update -y python3-ubs-engine*.rpm
sudo dnf update -y ubs-engine*.rpm
# 或
sudo rpm -Uvh ubs-engine*.rpm
sudo rpm -Uvh python3-ubs-engine*.rpm
```

### 5.2 卸载

```bash
# 单独卸载virt agent：
rpm -evh ubs-engine-virtagent
# 卸载ubs engine也会附带卸载virt agent：
systemctl stop ubse
sudo dnf remove -y ubs-engine*
# 如安装了插件包需先卸载插件：
rpm -evh ucache
rpm -evh rmrs
rpm -evh ubs-engine-virtagent
# 卸载UBSE所有包：
rpm -evh python3-ubs-engine
rpm -evh ubs-engine-client-devel
rpm -evh ubs-engine-client-libs
rpm -evh ubs-engine
```

> 卸载不会删除 <code>/var/lib/ubse/</code> 和 <code>/var/log/ubse/</code> 中的数据，如需清理请手动删除。
