# 构建指导

## 1 拉取源码

```shell
git clone ssh://git@szv-y.codehub.huawei.com:2222/BeiMing/service_domain/matrix_core/matrix_engine/ubs_engine.git
```

## 2 构建

### 2.1 构建依赖

Virt Agent作为UBS插件，构建依赖信息已经记录在UBS spec文件（ubs-engine.spec）中

具体内容如下：

```shell
BuildRequires: cmake make gcc-c++ gcc
BuildRequires: glibc-devel libstdc++-devel
BuildRequires: systemd-devel
BuildRequires: ubs-comm-devel
BuildRequires: numactl-libs
BuildRequires: bash bc coreutils findutils gawk grep kmod lsof libcap sed sudo util-linux vim util-linux-user cpio tar unzip mlocate ninja-build ccache java-11-openjdk python3-pre-commit libffi-devel zlib-devel clang-devel
Requires: bash glibc libgcc libstdc++ ubs-comm-lib
```

### 2.2 安装依赖

**方式一：DNF一键安装**

```shell
# 安装spec中指定的BuildRequires
cd ubs_engine
dnf builddep -y ubs-engine.spec
```

**方式二：yum手动安装**

```shell
yum install -y cmake make gcc-c++ glibc-devel libstdc++-devel systemd-devel audit-devel libboundscheck ubs-hcom-devel libxml2-devel numactl-libs bash bc coreutils findutils gawk grep kmod lsof libcap sed sudo util-linux vim util-linux-user cpio tar unzip mlocate ninja-build ccache java-11-openjdk python3-pre-commit libffi-devel zlib-devel clang-devel patchelf obmm
```

### 2.3 执行构建

```shell
# 执行 Release 构建（没有调试信息），产物输出到项目顶层目录的cmake-build-release/目录下
bash build.sh

# 执行 Debug 构建（附加调试信息），产物输出到项目顶层目录的cmake-build-debug/目录下
bash build.sh -D
```

## 3 形成rpm包

```shell
# 构建项目，并打包成 rpm 文件输出到项目顶层目录的 output/ 下。
bash build.sh package
```

产物如下：

```
└── output                                                          # 打包输出文件目录
    ├── ubs-engine-virtagent-<version>-<release>.aarch64.rpm            # 主程序包
    ├── python3-ubs-engine-<version>.rpm                           # python外层接口包
```
