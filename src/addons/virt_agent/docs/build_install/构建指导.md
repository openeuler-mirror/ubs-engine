# 构建指导

## 1 拉取源码

```shell
git clone https://gitcode.com/openeuler/ubs-engine.git
```

## 2 构建

### 2.1 构建依赖

Virt Agent作为UBSE插件，构建依赖UBSE。
[UBSE构建指导](../../../../../docs/build_install/构建指导.md/)

### 2.2 安装依赖

同UBSE安装依赖。
[UBSE构建指导](../../../../../docs/build_install/构建指导.md/)

### 2.3 执行构建

同UBSE构建。
[UBSE构建指导](../../../../../docs/build_install/构建指导.md/)

## 3 形成rpm包

同UBSE出包操作。
[UBSE构建指导](../../../../../docs/build_install/构建指导.md/)

Virt Agent产物如下：

```shell
└── output                                                          # 打包输出文件目录
    ├── ubs-engine-virtagent-<version>-<release>.aarch64.rpm            # 主程序包
    ├── python3-ubs-engine-<version>.rpm                           # python外层接口包
```
