# 构建指导

## 1 拉取源码

```shell
git clone https://gitcode.com/openeuler/ubs-engine.git
```

## 2 构建

### 2.1 构建依赖

Virt Agent作为UBSE插件，构建依赖信息已经记录在spec文件（ubs-engine.spec）中

### 2.2 安装依赖

UBSE及其依赖安装成功即可

### 2.3 执行构建

同UBSE构建。

## 3 形成rpm包

同ubse出包操作。

Virt Agent产物如下：

```
└── output                                                          # 打包输出文件目录
    ├── ubs-engine-virtagent-<version>-<release>.aarch64.rpm            # 主程序包
    ├── python3-ubs-engine-<version>.rpm                           # python外层接口包
```
