# 构建指导

## 1 拉取源码

```shell
git clone https://atomgit.com/openeuler/ubs-engine.git
```

## 2 构建

### 2.1 构建依赖

ubs-engine构建依赖信息记录在spec文件（[ubs-engine.spec](../../ubs-engine.spec)）中。
由于当前依赖的ubs-comm-devel版本只发布在openEuler 24.03 LTS SP3/SP4，通过`cat /etc/openEuler-release`确认编译系统版本：

- 如果编译系统版本是`openEuler release 24.03 (LTS-SP3)/(LTS-SP4)`，参考[2.1.1 openEuler release 24.03 (LTS-SP3)/(LTS-SP4) 依赖安装说明](#211-openeuler-release-2403-lts-sp3lts-sp4-依赖安装说明)
- 如果编译系统版本不是`openEuler release 24.03 (LTS-SP3)/(LTS-SP4)`，参考[2.2.2 其他openEuler版本依赖安装说明](#222-其他openeuler版本依赖安装说明)

#### 2.1.1 openEuler release 24.03 (LTS-SP3)/(LTS-SP4) 依赖安装说明

**方式一：DNF一键安装**

```shell
# 安装spec中指定的BuildRequires
cd ubs_engine
dnf builddep -y ubs-engine.spec
```

**方式二：yum手动安装**

```shell
yum install -y libboundscheck libxml2-devel numactl-libs openssl-devel rapidjson-devel patch 'cpp-httplib-devel >= 0.27.0' 'ubs-comm-devel >= 1.0.0-15'
```

#### 2.2.2 其他openEuler版本依赖安装说明

执行以下命令安装依赖：

```shell
yum install -y libboundscheck libxml2-devel numactl-libs openssl-devel rapidjson-devel patch 'cpp-httplib-devel >= 0.27.0'
```

从[openEuler软件仓库](https://dl-cdn.openeuler.openatom.cn/openEuler-24.03-LTS-SP3/)中下载ubs-comm-devel和ubs-comm-lib：

```shell
# 架构为aarch64
wget https://dl-cdn.openeuler.openatom.cn/openEuler-24.03-LTS-SP3/update/aarch64/Packages/ubs-comm-devel-1.0.0-15.oe2403sp3.aarch64.rpm
wget https://dl-cdn.openeuler.openatom.cn/openEuler-24.03-LTS-SP3/update/aarch64/Packages/ubs-comm-lib-1.0.0-15.oe2403sp3.aarch64.rpm
# 架构为x86_64
wget https://dl-cdn.openeuler.openatom.cn/openEuler-24.03-LTS-SP3/update/x86_64/Packages/ubs-comm-devel-1.0.0-15.oe2403sp3.x86_64.rpm
wget https://dl-cdn.openeuler.openatom.cn/openEuler-24.03-LTS-SP3/update/x86_64/Packages/ubs-comm-lib-1.0.0-15.oe2403sp3.x86_64.rpm
```

安装下载得到的rpm包：

```shell
rpm -ivh ubs-comm-*.rpm
```

### 2.3 执行构建

```shell
# 执行 Release 构建（没有调试信息，-O2 优化）
bash build.sh

# 执行 Debug 构建（附加调试信息）
bash build.sh -D

# 执行 RelWithDebInfo 构建（附加调试信息，-O2 优化）
bash build.sh -T RelWithDebInfo

# 执行 MinSizeRel 构建（没有调试信息，二进制文件最小构建，-Os 优化）
bash build.sh -T MinSizeRel
```

## 3 形成rpm包

```shell
# 构建项目，并打包成 rpm 文件输出到项目顶层目录的 output/ 下。
bash build.sh package
```

产物如下：

```
└── output                                                            # 打包输出文件目录
    ├── ubs-engine-<version>-<release>.aarch64.rpm                    # 主程序包
    ├── ubs-engine-client-libs-<version>-<release>.aarch64.rpm        # 客户端库
    ├── ubs-engine-client-devel-<version>-<release>.aarch64.rpm       # 客户端开发包
    └── ubs-engine-debuginfo-<version>-<release>.aarch64.rpm          # 调试信息
```
